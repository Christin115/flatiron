# JavaScript Calculator with History

This is a simple JavaScript calculator program that performs basic math operations and stores a history of calculations.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Input Validation 
- Display Calculation History

## How to Run

1. Make sure Node.js is installed.
2. Open a terminal in the project folder.
3. Run the following command:

node calculator.js

## Functions

### add(num1, num2)
Adds two numbers.

### subtract(num1, num2)
Subtracts two numbers.

### multiply(num1, num2)
Multiplies two numbers.

### divide(num1, num2)
Divides two numbers.

### displayHistory()
Displays all stored calculations.

### displayCalculation(index)
Displays a specfic calculation from the history
