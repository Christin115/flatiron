# Portfolio Platform

## Overview

Portfolio Platform is a Single Page Application (SPA) built with React and Vite that allows a creative agency to showcase projects in an organized and visually appealing way. Users can browse existing projects, search through the portfolio, and dynamically add new projects using a form.

The application demonstrates React fundamentals including component-based architecture, state management, event handling, and prop passing while providing a responsive user experience across devices.

---

## Features

* View a collection of portfolio projects
* Add new projects dynamically
* Search projects by title
* Responsive design for desktop, tablet, and mobile devices
* Reusable React components
* Project counter displaying the number of visible projects
* Empty state message when no projects match the search

---

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd portfolio-platform
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

5. Open your browser and visit

```text
http://localhost:5173
```

---

## Usage

### Search Projects

Use the search bar at the top of the page to filter projects by title.

### Add a Project

1. Complete the project form.
2. Enter:

   * Project Title
   * Category
   * Image URL
   * Description
3. Click **Add Project**.
4. The new project will immediately appear in the portfolio.
